# 收集品

- 收集品组件需要配合 [`收集品原型`](../sceneSettings/collectionPrototype) 使用
- 收集品组件需要配合 [`渲染器`](renderer) 组件及 [`触发器`](trigger) 组件使用
  - `渲染器` 组件负责图形渲染
  - `触发器` 组件负责触发收集品的收集
  - 建议直接通过 `创建 → 收集品` 来创建收集品

## `收集品名称`

- 类型：`string`
- 默认值：空

## `旋转速度`

- 类型：`Float3`
- 默认值：`(0, 1, 0)`

收集品在场景中旋转的速度。

## `音效`

- 类型：`Audio`
- 默认值：空

收集品被收集时播放的音效。
