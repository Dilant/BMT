# 贴花投射器

在一定范围内将贴花投射到物体上。

## [变换偏移](item#变换偏移)

## `材质`

- 类型：`Material`
- 默认值：空

贴花的材质。参考 [素材 → 材质 → 材质属性 → 贴花材质](../assets#材质属性) 。

## `绘制距离`

- 类型：`float`
- 默认值：`50.0`

当摄像机和该组件的距离大于此值时，贴花将不会被渲染。

## `开始淡出比率`

- 类型：`float`
- 默认值：`0.8`

当摄像机和该组件的距离大于 `此值 * 绘制距离` 时，贴花将开始淡出。

## `不透明度`

- 类型：`float`
- 默认值：`1.0`

## `旋转速度`

- 类型：`float`
- 默认值：`0.0`
