# 场景设置 {#scene-settings}

- `场景设置` 每张地图有且仅有一个
- `场景设置` 不可被创建，不可被删除

## 组件 {#components}

`场景设置` 拥有以下 13 个组件：

- [`地图信息`](mapInfo)
- [`环境`](environment)
- [`光照`](lighting)
- [`天空`](sky)
- [`云`](cloud)
- [`雾`](fog)
- [`镜头`](lens)
- [`背景音乐`](backgroundMusic)
- ~~[`地形`](terrain)~~
- [`收集品原型`](collectionPrototype)
- [`素材引用`](assetReference)
- [`自定义玩家球`](customBall)
- [`造景`](landscaping)
